    • » html	表示结构
    • » css	表示样式

  • JS三大核心 
    • » ECMAScript
      • JS的标准，语法
    • » BOM(Browser Object Model)
      • 一整套操作浏览器的属性和方法
    • » DOM(Documentv Object Model)
      • 一整套操作文档流的属性和方法

      • 文档对象模型 (DOM) 是HTML和XML文档的编程接口。它提供了对文档的结构化的表述，并定义了一种方式可以使从程序中对该结构进行访问，从而改变文档的结构，样式和内容。
        DOM 将文档解析为一个由节点和对象（包含属性和方法的对象）组成的结构集合。简言之，它会将web页面和脚本或程序语言连接起来。
